# P5 Sports Physio Website

Static website for P5 Sports Physio.

## How to open locally

Open `index.html` in a browser.

## Main files

- `index.html` - website page
- `assets/` - logo, hero image, and supporting club logos

## Notes

- Online bookings are embedded through Cliniko.
- The enquiry form currently uses a `mailto:` email action.
- Do not commit Cliniko passwords, API keys, or private credentials.
